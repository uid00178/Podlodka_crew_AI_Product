| Файл | Что это |
|---|---|
| `CLAUDE_template.md` | Шаблон контекста продукта для AI-аналитика |

## Минимальный воркфлоу для первой проверки гипотезы

1. Заполните `CLAUDE_template.md` под свой продукт
2. Прогоните идею через `skill-hypothesis-check.md` — получите чёткую гипотезу + ICE
3. Сделайте 1 синтетический CustDev (`skill-synthetic-custdev.md`) — найдите дыры
4. Сделайте `skill-market-scan.md` по нише — увидите рынок
5. Запишите вывод в `skill-decision-log.md` — фиксируем решение
6. Положите decision log обратно в `CLAUDE.md` — петля замкнулась

Один цикл = ~3 часа вашего времени вместо недели.

## Что НЕ делает этот набор

- Не заменяет реальный CustDev — он только готовит к нему
- Не принимает решения за вас — даёт draft, вы корректируете
- Не работает без вашего суждения — AI не валидирует свои ответы
